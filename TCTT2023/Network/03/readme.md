# Write-ups for TCTT2023/Network/03

## Flag pattern

`CTT23{xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx}`

## Challenge Files

[Networking-MitM.zip](./Networking-MitM.zip)

## Solution

1. First, Let open the pcap file with wireshark and analyze it with `Statistics > Protocol Hierarchy`.

![wireshark_result_1](./write-ups/01.png)

![wireshark_result_2](./write-ups/02.png)

2. It look like there is a lot of SMTP request, but most of the time it's a bruteforce username and password. There is only packet `8130` that successfully login.

![wireshark_result_3](./write-ups/03.png)

with username and password `QWRtaW5TTVRQ : V293IVlvdUZvdW5kUGFzc3dvcmQ=`, and when we decode with `BASE64` we got `AdminSMTP : Wow!YouFoundPassword`

3. We follow the TCP stream of the packet `8130` and we got the email content.

![wireshark_result_4](./write-ups/04.png)

![wireshark_result_5](./write-ups/05.png)

4. We can see that the email content is contain docx file named secret.docx, which is base64 encoded. So we copy the content.
